##  常见标签

###  h1~h6

标题标签，字体加粗

###  hr

折行标签，有折线

###  p

段落标签

### div

布局标签

###  span

行内元素标签

### a

链接标签，锚点（#），target

###  ul  ol dl

列表标签，ul：无序列表，ol：有序列表，ol：定义列表

###  input

输入框，password：密码，mail：邮箱，text：文本，submit：提交，button：按钮，number：数字。。。

###  img

图像标签 ，src   alter

###  table

表格，th：标题，td：内容
