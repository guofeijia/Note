##		ls

ls是最常见的目录操作命令，主要作用是显示目录下的内容

* 命令名称：ls。
*  英文原意：list。
*  所在路径：/bin/ls。
*  执行权限：所有用户。
*  功能描述：显示目录下的内容。

###		命令格式

``` 
[root@localhost ~]#ls [选项][文件名或目录名]
```

选项：

* -a：显示所有文件；
* --color=when：支持颜色输出，when 的值默认是 always（总显示颜色），也可以是 never（从不显示颜色）和 auto（自动）；
* -d：显示目录信息，而不是目录下的文件；
* -h：人性化显示，按照我们习惯的单位显示文件大小；
* -i：显示文件的 i 节点号；
* -l：长格式显示；

ll 查看目录下文件详细信息

* -a ：全部
* -S：按文件大小排序
* -t：按时间排序
* -r：排序反转
* -d：查看目录本身，不列出目录下信息
